⏺ 你说得对，"有开单就行"等于"活的就行"，门槛太低。                              
                                                                              
  按正常逻辑想：                                                                
   
  一个策略如果在跑，最起码多久开一单？                                          
                                                                             
  保守说一个月至少开几单吧。如果一个月都不开一单，这策略等于半死不活。

  按"至少每两周开一单"来算：

  ┌──────┬──────────┬──────────┬────────┐
  │ 周期 │ 数据年数 │ 每年最少 │ 总门槛 │
  ├──────┼──────────┼──────────┼────────┤
  │ 1m   │ ~2年     │ 26单/年  │ ~50    │
  ├──────┼──────────┼──────────┼────────┤
  │ 5m   │ ~4年     │ 26单/年  │ ~100   │
  ├──────┼──────────┼──────────┼────────┤
  │ 15m  │ ~4年     │ 26单/年  │ ~100   │
  ├──────┼──────────┼──────────┼────────┤
  │ 1h   │ 5~8年    │ 26单/年  │ ~150   │
  └──────┴──────────┴──────────┴────────┘

  如果觉得两周一单太松，按"至少每周一单"：

  ┌──────┬──────────┬──────────┬────────┐
  │ 周期 │ 数据年数 │ 每年最少 │ 总门槛 │
  ├──────┼──────────┼──────────┼────────┤
  │ 1m   │ ~2年     │ 52单/年  │ ~100   │
  ├──────┼──────────┼──────────┼────────┤
  │ 5m   │ ~4年     │ 52单/年  │ ~200   │
  ├──────┼──────────┼──────────┼────────┤
  │ 15m  │ ~4年     │ 52单/年  │ ~200   │
  ├──────┼──────────┼──────────┼────────┤
  │ 1h   │ 5~8年    │ 52单/年  │ ~300   │
  └──────┴──────────┴──────────┴────────┘

  这是按"正常活跃度"推的，不是按数据分布推的。你觉得哪个频率更合理？每周一单还是
  两周一单？或者你有别的标准？
