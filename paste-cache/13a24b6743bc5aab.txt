Hotfix 2：K 线历史重连的绝对线程安全
在 feed_gateway.py 的 _load_history 中，为 = 赋值加上微秒级锁，防漏 K 线：

Python
    def _load_history(self, api_key, api_secret, testnet):
        adapter = BitMEXAdapter({'api_key':api_key,'api_secret':api_secret,'testnet':testnet})
        adapter._ws = None; adapter._connected = True
        for c in self.coins:
            iv = self.intervals[c]; key = f"{c}_{iv}"
            klines = adapter.get_klines(c, iv, limit=250)
            if klines:
                new_window = deque(klines, maxlen=250)
                # 【Hotfix关键】：加锁赋值，杜绝极小概率的数据竞争
                with self._lock:
                    self._kline_windows[key] = new_window
                logger.info(f"  {key}: 加载 {len(klines)} 根 K 线")
