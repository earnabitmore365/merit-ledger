# Claude Code 自定义配置备份

## 内容

| 目录 | 说明 |
|------|------|
| `.claude/settings.json` | 全局配置 — DeepSeek API、hooks、权限、插件 |
| `.claude/CLAUDE.md` | 全局指令 — 无极堂规 |
| `.claude/scripts/` | 自定义 hooks 脚本 — 会话管理、记忆同步、监控 |
| `.claude/mcp/` | MCP 服务器 — 交易系统运维(wuji_ops)、Pine Script(pine-mcp) |
| `cashflowAPP/.claude/agents/` | Agent 定义 — data-agent、ui-agent、qa-agent |
| `cashflowAPP/.claude/skills/` | Skills — database、services、ui-components、ui-pages、qa-verify、orchestrator |

## 用途

重装 Claude Code 后恢复自定义配置时使用。
